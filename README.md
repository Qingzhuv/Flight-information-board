# ✈️ WESTCRAN 航班动态显示系统 (V1.0)

本项目是一个专为机场、候机室或相关业务展示设计的航班动态看板系统。支持全屏大屏展示、移动端自适应管理后台。

**技术支持：** [WESTCRAN 西鹤软件](https://www.westcran.tech)

**开源协议：** GPL-3.0 

------

## 🛠️ 环境要求

- **服务器：** 支持 PHP 7.4+ 的 Web 服务器（Nginx / Apache / IIS）
- **数据库：** MySQL 5.7+
- **网络环境：** 支持离线部署（所有静态资源已本地化）

------

## 🚀 快速部署步骤

### 1. 下载与解压

将本项目代码下载到你服务器的 Web 根目录。

### 2. 初始化数据库

1. 登录你的 MySQL 管理工具（如 phpMyAdmin 或 Navicat）。
2. 创建一个新的数据库，命名为 `flight_db`。
3. 运行init_db.php（如果没有，请手动执行以下 SQL）：

### 3. 配置数据库连接

打开 `config.php` 文件，修改以下配置以匹配你的服务器环境：

PHP

```php
$db_host = 'localhost';
$db_user = '你的数据库用户名';
$db_pass = '你的数据库密码';
$db_name = 'flight_db';
```

### 4. 权限设置

确保 `assets/` 文件夹具有读取权限。如果遇到登录失效，请检查服务器的 `session` 存储目录是否可写。

------

## 📂 访问说明

- **前台大屏展示：** `http://你的域名/index.php` (建议按 F11 开启浏览器全屏)
- **后台管理系统：** `http://你的域名/admin.php`
- **默认管理员账号：** `admin`
- **默认管理员密码：** `123456` (请务必在 `config.php` 或数据库中修改)
