# ✈️ WESTCRAN 航班动态显示系统 (V1.0)

本项目是由 **[WESTCRAN 西鹤软件](https://www.westcran.tech)** 开发的轻量级航班看板解决方案。适用于架空虚拟机场的实时信息展示。

## ✨ 核心特性

- **全屏看板**：针对大屏优化，支持浏览器全屏模式。
- **移动端适配**：管理后台采用响应式设计，支持手机随时随地发布航班。
- **本地化部署**：静态资源（Bootstrap/jQuery）已完全本地化，支持无互联网环境运行。
- **实时更新**：基于异步请求，大屏数据自动刷新，无需手动干预。

## 🛠️ 环境要求

- **服务器**：支持 PHP 7.4+ 的 Web 服务器（Nginx / Apache / IIS）
- **数据库**：MySQL 5.7+
- **网络环境**：支持离线部署（所有静态资源已本地化）

## 🚀 快速部署步骤

### 1. 下载与解压

将本项目代码下载并解压至服务器的 Web 根目录。

### 2. 初始化数据库

1. 登录 MySQL 管理工具（如 phpMyAdmin 或 Navicat）。
2. 创建数据库 `flight_db`，字符集推荐 `utf8mb4`。
3. 执行根目录下的 `init_db.php` 脚本（或手动导入 SQL 语句）。

### 3. 配置数据库连接

打开 `config.php` 文件，修改以下配置：

PHP

```php
$db_host = 'localhost';
$db_user = '你的数据库用户名';
$db_pass = '你的数据库密码';
$db_name = 'flight_db';
```

## 📂 访问说明

- **前台大屏展示**：http://你的域名/index.php
- **后台管理系统**：http://你的域名/admin.php
- **默认管理员账号**：admin
- **默认管理员密码**：123456 (请务必在登录后立即修改)
